Sitio Seguro Local

Este es un ejemplo de sitio seguro configurado con firma SSL y servido por Apache2.

Generado desde un Dockerfile, provisionado por .sh con las siguientes acciones:

  • Copiado de los archivos .conf al directorio /etc/apache2/external/ creado previamente por el Dockerfile
  • Copiado del contenido de la carptea HTML a el directorio /var/www/html del Docker
  • Generación de CRT, CSR, KEY
  • Ejecución de comando CMD definido el el DockerFile para reinicar el Apache
